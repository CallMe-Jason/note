# 作用

ajax 主要是实现页面和 web 服务器之间数据的异步传输。

---

## 客户端与服务器

负责获取和消费资源的电脑，就叫做客户端

---

## URL地址的概念

称为统一资源定位符。

组成：

1.通信协议

2.服务器名称

3.在服务器上具体的存放位置

```js
http
//通信协议
www.cnblogs.com
//服务器名称
asdad/p/1232523.html
//服务器的存放位置
```

---

## 资源请求的方式

最常见的请求方式分为get和post

get请求通常用于获取服务器资源，如地址，HTML文件等

post用于向服务器发送资源，如用户登录信息

## jQuery中的Ajax

```js

$.post()//给服务器发送
$.ajax()//都可
```


---

## $.post()

```js
$.post(url,[data],[callback])
```

![$.post](笔记截图/$.post.png)

---

## $.ajax()

```js
$.ajax({
//type可以不写，默认就是get
	type:'get',
	url:'http://www.liulongbin.top:3006/api/getbooks',
//data可以不写，上传数据时要写
	data:{id:1},
	success : (res) => console.log(res)
})
```
